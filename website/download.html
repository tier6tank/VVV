VVV current version is 1.1 and it is available from the Sourceforge download page. That page shows all the available files, so it might be confusing.

To help users this page contains direct links and some information for each operating system.

Windows


Click here to download the Windows version. It is packaged as an installer to make things easier. The installer simply copies the files to the desired folder and it associates .vvv files to the program so that it is possible to open catalogs double clicking them.

You can also download a zip file that does not need to be installed. Just expand the archive and run vvv.exe.

Linux


Click here to download the 32 bit Linux version. The archive contains installation instructions.

Click here to download the 64 bit Linux version. The archive contains installation instructions.

OS X


Click here to download the OS X version (INTEL ONLY). You will download a dmg file: just save it, open it and copy vvv.app to your Applications folder. You can also drag the vvv.app icon and drop it into the nearby Applications icon to easily copy it.
